# **App Name**: Computer Plus

## Core Features:

- Navbar: Implement a responsive navbar with logo, quick links (Home, Order, Report, Support), and account dropdown with 'Account Settings' and 'Logout' options.
- Category Section: Implement a category section with a dropdown for 'All Categories' and featured category links.
- Product Section: Develop a product section featuring a search bar, product listing with pagination, and individual product cards.
- Product Card Layout: Design a vertical product card with a 1:1 image area, title, price, discount badge (top-left), and New Arrival badge (top-right).
- Footer: Create a footer with legal links (Shipping Policy, Return Policy, About Warranty, Terms and Conditions, Privacy Policy) and social media icons (Instagram, Facebook, WhatsApp) with a copyright line.

## Style Guidelines:

- Background: Neutral light gray (#FAFAFA) for a clean, breathable aesthetic.
- Primary Color: Soft pink (#F472B6) to represent the soft red to soft pink gradient requested.
- Accent: Slightly brighter pink (#F0297B) used for buttons, links, and active states.
- Headings: 'Bricolage Grotesque' (sans-serif) for headings.
- Body: 'Plus Jakarta Sans' (sans-serif) for body text.
- Minimal icons from Lucide Icons library.
- Card-driven layout with rounded corners (rounded-2xl).
- Subtle animations (opacity, scale, translate-y) with 150ms-300ms duration using Tailwind's default easing.
